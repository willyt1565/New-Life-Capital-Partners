# Brand Assets — Drop Files Here

This folder is where you upload the real brand assets. The site is built with
clearly-marked placeholders wherever an asset is missing, so you can swap things
in gradually. **File names matter** — match the names below and the site picks
them up automatically with no code changes.

## /logos
| File name | Used for |
|---|---|
| `nlcp-logo.svg` (or `.png`) | New Life Capital Partners logo — nav + footer |
| `high-hopes-logo.svg` (or `.png`) | High Hopes Home Buyers — Our Companies card |
| `newera-logo.svg` (or `.png`) | Newera Property Management — Our Companies card |

> SVG is strongly preferred for logos (crisp at any size, tiny file). PNG with a
> transparent background is the fallback.

## /photography
Dark, moody property photography — aerial shots, architectural interiors,
well-lit empty spaces. **No bright stock photos of smiling families.**

| File name | Used for | Recommended size |
|---|---|---|
| `hero.jpg` | Homepage hero background | 2400×1400, < 400 KB after compression |
| `who-we-are.jpg` | "Who We Are" section image | 1600×1200 |
| `companies-bg.jpg` | Optional texture behind company cards | 2000×1200 |

## /headshots
Leadership team headshots — professional, shot on or processed to a dark
background so they sit naturally in the site.

| File name | Used for |
|---|---|
| `leader-1.jpg`, `leader-2.jpg`, ... | About page leadership grid |

Recommended: square crop, 800×800, consistent lighting across all headshots.

## /icons
Reserved for any custom iconography. The site currently uses inline SVG line
icons (gold), so you only need to add files here if you want to replace them.

---

### Image optimization
Before uploading photography, compress it (TinyPNG, Squoosh, or ImageOptim).
Target: homepage loads fully in under 3 seconds. Large unoptimized photos are
the single most common cause of a slow luxury site.
